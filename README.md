# landingpage-learning1
<h3><strong>Projeto de Landing Page para Academia, utilizando apenas Html e Css</strong></h3>

<h3><strong>Descrição:</strong></h3>
O objetivo deste projeto é criar uma Landing Page para uma academia, com o intuito de promover os serviços oferecidos e atrair potenciais clientes. A Landing Page é uma página web única e direcionada, focada em conversão, que apresenta informações relevantes de forma a captar o interesse do usuário e incentivá-lo a realizar uma ação específica, como fazer uma inscrição, agendar uma visita ou entrar em contato.

<h3><strong>Funcionalidades:</strong></h3>
Apresentação da academia: A Landing Page deve destacar os principais informações da academia, como localização, horários de funcionamento, variedade de aulas, profissionais qualificados, etc.

Formulário de contato: Um formulário de contato deve ser disponibilizado para que os visitantes possam enviar suas dúvidas, solicitar mais informações ou agendar uma visita. Os campos básicos como nome, e-mail e mensagem devem estar presentes.

Integração com redes sociais: Incluir links para as redes sociais da academia, permitindo que os visitantes acompanhem as novidades e interajam com a academia nas mídias sociais.

Call to action: Adicionar botões de call to action estrategicamente posicionados ao longo da página, incentivando os visitantes a realizar a ação desejada, como agendar uma aula experimental, fazer uma inscrição ou entrar em contato.

Depoimentos de clientes: Incluir seção com depoimentos de clientes satisfeitos, destacando suas experiências positivas com os serviços da academia.

<h3><strong>Tecnologias Utilizadas:</strong></h3>
HTML5: Linguagem de marcação utilizada para estruturar a página web.
CSS3: Utilizado para estilizar e dar vida à página.
